## 🌟 Overview

**Shah Ji Wooden & Home Decor** is a bespoke portfolio and service website crafted for a premium interior design studio serving Delhi & Uttarakhand. As a freelance project, this platform was custom-built from the ground up to reflect the brand's luxurious and handcrafted approach to interior transformation, combining stunning visual storytelling with a flawless user experience.

The site is engineered to convert visitors into clients by offering deep dives into completed projects, detailed breakdowns of their 6-step collaborative process, and seamless WhatsApp integration for instant consultations.

## ✨ Key Features

- **Immersive Visual Experience**: Smooth scroll using Lenis, micro-interactions with Framer Motion, and a crafted splash screen that builds anticipation.
- **Dynamic Project Gallery**: Editorial-style layouts showcasing luxury transformations, modular kitchens, and residential designs using Embla Carousel.
- **Blazing Fast Performance**: Next.js App Router utilization with optimized WebP assets, strict Image component handling, and Vercel Speed Insights tailored for peak performance.
- **Conversion-Optimized UX**: Persistent, floating "WhatsApp Concierge" and strategic CTA placements driving direct mobile connections.
- **Fully Responsive Architecture**: Mobile-first approach guaranteeing pixel-perfect styling from small mobile screens up to ultra-wide displays.
- **Modern Tech Stack Integration**: Seamless animations and complex UI states handled flawlessly without sacrificing Core Web Vitals.

## 🛠️ Technology Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Directory)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & PostCSS
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Scroll Management**: [Lenis](https://lenis.studiofreight.com/)
- **Carousels**: [Embla Carousel React](https://www.embla-carousel.com/)
- **Analytics & Tracking**: Vercel Analytics & Speed Insights
- **Image Optimization**: Sharp & Next.js Image Component

## 🚀 Case Study & Development Journey

Developed as a freelance undertaking, the primary goals for this project included migrating an aging web presence to a modern, lightning-fast architecture. Key development milestones included:
- **Refining Mobile UX**: Solving complex text overflows and optimizing the hero section specifically for high conversion on mobile devices.
- **Gallery & Asset Optimization**: Converting heavy payload imagery to optimized WebPs, resulting in drastic reductions in Time to Interactive (TTI).
- **SEO & Search Dominance**: Configuring robust static generation, structured metadata, and rectifying 404 indexing errors to ensure optimal search engine rankings.
